# YouTube Trending Video Analysis

## Project Overview
This project analyzes trending YouTube videos using data analytics and visualization techniques. The goal is to identify patterns, trends, and key factors that influence video popularity.

## Dataset
- Trending YouTube Video Statistics dataset
- Category mapping file: CA_category_id.json :contentReference[oaicite:0]{index=0}

## Objectives
- Analyze trending video patterns
- Identify most popular categories
- Study engagement metrics (views, likes, comments)
- Understand factors influencing video trends

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis Performed
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Category-wise Analysis
- Engagement Metrics Analysis
- Trend Visualization

## Sample Insights
- Entertainment and Music categories dominate trending videos
- Higher engagement (likes/comments) correlates with higher views
- Titles and tags play a crucial role in visibility

## Future Improvements
- Build predictive model for trending videos
- Deploy dashboard using Streamlit
- Perform sentiment analysis on titles and comments
